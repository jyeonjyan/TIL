## nest 빠르게 환경 구성하기

### Installation
> https://docs.nestjs.kr/

nest CLI 를 사용하면 빠르게 프로젝트를 스캐폴딩하거나 시작 할 수 있습니다.

1. Global install nestjs cli
```
$ npm i -g @nestjs/cli
```

2. get start nest project
```
$ nest new project-name
```

이런 방법도 있습니다 **Git**으로 Ts 시작 프로젝트를 설치하려면..
```
$ git clone https://github.com/nestjs/typescript-starter.git project
$ cd project
$ npm install
$ npm run start
```
